## Kaj je Ghostcore Evidence OS V6

Ghostcore Evidence OS V6 je operacijski sistem za gradnjo forenzičnih dokaznih paketov iz YAML definicij. 

Iz enega `case.yaml` datoteke sistem generira:
- PDF (profesionalen evidence report)
- DOCX (distributivni dokument za track changes)
- HTML/Markdown (spletna verzija)
- ZIP (celoten evidence paket)
- JSON metadata (identiteta in zaščita)

---

## Kanonični Case

Trenutno aktivni case: `epstein_tech_001.yaml`

**Naslov:** "Epstein–Tech–Government Nexus: Forensic Briefing"  
**Identiteta:** Šabad (Independent Researcher)  
**Zaščita:** NO SPECULATION. ONLY SOURCED EVIDENCE.

Case vsebuje:
- Executive Summary
- Methodology  
- Verified Findings
- Structural Patterns
- Timeline of Events
- Reform Pathways

---

## Kako uporabiti

1. Pripravi case v YAML formatu v `/cases/`
2. Poženi kompilacijo:

```bash
./bin/ghostcore build cases/epstein_tech_001.yaml
```

3. Rezultati se nahajajo v `/build/`

---

## Arhitektura

- `/core/` - kernel, sync, preview, AI, identity
- `/modules/` - PDF/DOCX/ZIP generatorji  
- `/cases/` - FCM definicije
- `/build/` - izhodni paketi
- `/bin/` - CLI dostop
