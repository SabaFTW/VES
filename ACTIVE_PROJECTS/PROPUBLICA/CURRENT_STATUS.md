# Template_Evidence_Log.md

To je **osnovni evidence/fact-check log**, inspiriran po ProPublica-style “every fact has a row”.

Vsak case naj ima svojo kopijo te datoteke:
- npr. `EVID_DIGITAL_CAGE.md`, `EVID_CASE_X.md`, itd.

---

## 1. NAVODILA

- Vsak pomemben **fact** (trditev, številka, citat) dobi svojo vrstico.
- “Source 1” in “Source 2” sta ne nujno dve različni osebi/dokumenta – lahko je:
  - dokument + uradna baza podatkov,
  - intervju + dokument,
  - dva neodvisna dokumenta, itd.
- “Status verifikacije” naj bo minimalno: `osnutek / delno / potrjeno`.

## 4. NAVODILA ZA AGENTE

Ko agent pomaga pri konkretnem case-u:

1. Naj NE piše dokazov iz lufta – referencira naj obstoječe datoteke / transkripte / dokumente.  
2. Če datoteka ne obstaja, lahko pusti placeholder (npr. `TODO: FOIA request X`).  
3. Naj raje doda nove vrstice, kot pa da prepisuje ali briše stare (razen če mu ti to izrecno naročiš).  
4. Če agent generira povzetke, naj jasno označi, kaj je:
   - **dejanski cite** iz dokumenta/intervjuja,
   - **interpretacija / analiza** (da se ne meša fact in komentar).
