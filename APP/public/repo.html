VES Repository Map

This page provides an overview of major directories in the VES repository. Click a directory to view its contents on GitHub.

  • .github – GitHub configuration and workflow files.
  • ACTIVE_FLAME – Scripts and services for real-time processes.
  • APP – Source code and public files for the GhostCORE portal.
  • ARCHIVE – Archived documents and past iterations.
  • ASSETS – Images, icons and multimedia assets.
  • CONSCIOUSNESS_LAB – Research notes and analytical experiments.
  • DATA – Data sets and structured information.
  • DOCS – Documentation and technical papers.
  • GHOST_OS – GhostOS systems and modules.
  • GRIMORIJ_ARA – Narrative and creative writings.
  • JOURNALS – Journals and logs.
  • RESONANCE – Resonance-based analysis and prototypes.
  • SERVICES – Service APIs and backend components.
  • TROP – Tropical files and experiments.
  • VORTEX – Work related to the Vortex module.
  • foundation_repo – Neutral archival and analysis workspace.
