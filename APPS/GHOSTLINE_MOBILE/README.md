### Prerequisites

* Node.js (v18+)
* npm (v9+)

### Local Development

1. **Install Dependencies**

    ```bash
    npm install
    ```

2. **Start Development Server**

    ```bash
    npm run dev -- --host
    ```

3. **Access on Mobile**
    * Ensure your phone is on the same Wi-Fi network.
    * Check the terminal output for the local IP address (e.g., `http://192.168.1.X:5173`).
    * Open that URL in your mobile browser.
