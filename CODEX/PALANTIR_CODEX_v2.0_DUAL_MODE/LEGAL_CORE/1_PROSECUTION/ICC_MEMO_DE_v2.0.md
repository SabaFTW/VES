# DE Rechtsgutachten v2.0

## Zusammenfassung

Dieses Gutachten behandelt die juristische Grundlage für die Verfolgung von Unternehmen, die in das Lavender/Gospel-System und seine zivilen Anwendungen durch die Chat Control-Architektur verwickelt sind. Der Nachweis führt systematische Verstöße gegen das internationale humanitäre Recht, die Unternehmensbeteiligung an Kriegsverbrechen und grenzüberschreitende kriminelle Aktivitäten.
