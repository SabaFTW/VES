# SLO Pravna analiza v2.0

## Povzetek

Natančno dokumentiranje primerjalne analize med vojaško aplikacijo Lavender/Gospel sistema in civilno aplikacijo Chat Control arhitekture. Dokazila prikazujejo sistematične kršitve mednarodnega humanitarnega prava, korporativno povezanost s kaznivimi dejanji ter transnacionalno kriminalno aktivnost.
