# Discovery Requests

## Formal Legal Requests for Corporate Documentation

### Request to Palantir Technologies Inc. (Case No. ICC-2025-001-LAVENDER)

1. All internal communications regarding Project Lavender (2022-present)
2. Financial records showing payments from Israeli Defense Forces/Unit 8200
3. Technical documentation of AI targeting algorithms used in Gaza
4. Personnel records for individuals who worked on military contracts transferred to civilian projects
5. Partnership agreements with Thorn/NCMEC and EU Commission
6. Patent applications for civilian applications of military AI systems

### Request to European Commission (Case No. ICC-2025-002-CHAT)

1. All communications regarding Chat Control architecture implementation
2. Funding agreements with Palantir and Thorn SA
3. Personnel exchanges between military contractor alumni and EU institutions
4. Technical specifications for client-side scanning implementations
5. Risk assessments for human rights violations in proposed regulations

### Request to Thorn SA (Case No. ICC-2025-003-SAFTER)

1. All technical documentation for Safer AI system
2. Communications with Palantir regarding AI architecture sharing
3. Records of data sharing with law enforcement agencies
4. Technical papers on hash collision probabilities in CSAM detection
5. Personnel records for Unit 8200 alumni employed at Thorn
